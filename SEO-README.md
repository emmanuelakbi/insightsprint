## 📋 What You Need to Do

### 1. Create Images (Required)

Create these 5 images in the `/public` folder:

| File                   | Size       | Purpose              |
| ---------------------- | ---------- | -------------------- |
| `og-image.png`         | 1200x630px | Social media sharing |
| `favicon.svg`          | 32x32px    | Browser tab icon     |
| `apple-touch-icon.png` | 180x180px  | iOS home screen      |
| `icon-192.png`         | 192x192px  | PWA icon             |
| `icon-512.png`         | 512x512px  | PWA icon (high-res)  |

**Design Guidelines:**

- Colors: #121212 (Ink), #FF6B35 (Accent), #FFFFFF (White)
- Style: Bold, brutalist, high contrast
- Text: Large and readable

### 2. Update URLs (Required)

Replace `insightsprint.vercel.app` with your actual domain in:

- `index.html`
- `public/sitemap.xml`
- `public/robots.txt`

### 3. Set Up Analytics (After Launch)

- Create Google Analytics account
- Add tracking code to `index.html`
- Set up Google Search Console
- Submit sitemap
